Sobre o TabStats

O TabStats é um site criado para ver informações de outros usuários do TabNews

O projeto também é open-source e foi desenvolvido usando HTML5, CSS3 e JavaScript (puros), hospedado na Netlify e usando a API aberta do TabNews. Você pode acessar o repositório Clicando aqui, sua contribuição será muito bem-vinda!

Criado especialmente para uma comunidade incrível.
